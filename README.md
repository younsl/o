# o

[Observability](https://opentelemetry.io/docs/concepts/observability-primer/) and [Kubernetes](https://github.com/kubernetes/kubernetes) operation addons built with [Rust](https://github.com/rust-lang/rust) [1.95+](https://github.com/rust-lang/rust/releases/tag/1.95.0). Includes a [blog](./blog/) ([Zola](https://github.com/getzola/zola)), CLI [tools](./box/tools/), and [kubernetes](./box/kubernetes/) addons, operators, and runtime images.
