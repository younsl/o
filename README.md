# o

A collection of [observability](https://opentelemetry.io/docs/concepts/observability-primer/) and [Kubernetes](https://github.com/kubernetes/kubernetes) cluster operation addons.

Built with [Rust](https://github.com/rust-lang/rust) [1.95+](https://github.com/rust-lang/rust/releases/tag/1.95.0), these tools deliver operational advantages: minimal container sizes, low memory footprint, static binaries with no runtime dependencies, memory safety, and compile-time guarantees for production stability.

## Contents

Production-ready tools, controllers, and automation resources for [Kubernetes](https://github.com/kubernetes/kubernetes) and cloud infrastructure.

- **[blog](./blog/)** - Tech blog powered by [Zola](https://github.com/getzola/zola)(a fast static site generator written in [Rust](https://github.com/rust-lang/rust)), deployed to GitHub Pages
- **[tools](./box/tools/)** - CLI tools for monitoring and testing
- **[kubernetes](./box/kubernetes/)** - Kubernetes addons, operators, and runtime images
