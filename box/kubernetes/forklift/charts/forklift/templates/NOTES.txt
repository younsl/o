High availability:
{{- if .Values.replication.enabled }}
  PV-BASED REPLICATION is ENABLED ({{ .Values.replicaCount }} replicas, StatefulSet, per-pod RWO PVC).
  The standby pulls the leader's data every {{ .Values.replication.interval }}; writes within
  one interval can be lost on failover. Traffic follows the forklift.io/role=leader pod label.
{{- else if eq (include "forklift.haEnabled" .) "true" }}
  HA is ENABLED ({{ .Values.replicaCount }} replicas, leader election via Lease "{{ include "forklift.leaseName" . }}").
  Ensure persistence.accessModes is ReadWriteMany so all replicas share the volume.
  No RWX storage available? Set replication.enabled=true for per-pod RWO volumes.
{{- else }}
  HA is DISABLED (single instance). Set replicaCount > 1 (and RWX storage, or
  replication.enabled=true for per-pod RWO volumes) for HA.
{{- end }}

  open http://localhost:8080
