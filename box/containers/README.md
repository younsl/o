# Container Images

Custom container images hosted on [ghcr.io](https://github.com/younsl?tab=packages). Inspired by [bitnami/containers](https://github.com/bitnami/containers).

- [backstage](./backstage/) — Internal Developer Portal (Node.js)
- [filesystem-cleaner](./filesystem-cleaner/) — Sidecar container for directory monitoring and cleanup (Rust)
- [logstash-with-opensearch-plugin](./logstash-with-opensearch-plugin/) — Logstash with OpenSearch output plugin for ECK (JVM)
